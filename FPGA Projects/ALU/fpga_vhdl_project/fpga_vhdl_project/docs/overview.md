# ALU Project Overview

## Purpose
Synchronous 8-bit Arithmetic Logic Unit (ALU) with support for eight operations and status flags, designed for educational and prototyping use on generic FPGA targets.

## Features
- Operations: ADD, SUB, AND, OR, XOR, NOT, SLL, SRL, NOP
- Flags: Zero flag, Carry/Borrow flag
- Configurable data width via generic (default 8 bits)
- Synchronous active-high reset
- GHDL-simulatable RTL and self-checking testbench
