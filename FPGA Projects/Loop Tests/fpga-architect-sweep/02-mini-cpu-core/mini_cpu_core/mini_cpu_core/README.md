# Mini CPU Core
A minimal 8-bit CPU core designed for VHDL-2008 and GHDL simulation.

## Features
- Fetch/Decode/Execute pipeline.
- ALU with ADD, SUB, AND, OR operations.
- 8 registers and 256-byte memory interfaces.
- Self-checking testbench with assertions.

## Usage
Run `make` to compile and simulate.
Check `cpu_core_tb.vcd` for waveforms.
