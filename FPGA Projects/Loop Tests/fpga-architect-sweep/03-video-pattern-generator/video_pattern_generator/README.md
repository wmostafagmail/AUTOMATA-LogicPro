# Video Pattern Generator

## Overview
Generates VGA-compatible timing and RGB565 color bars using a clean, synchronous architecture.

## Files
- `src/`: VHDL RTL sources
- `tb/`: Self-checking testbench
- `constraints/`: XDC constraints placeholder
- `Makefile`: GHDL build/sim automation
- `sim/run_ghdl.sh`: Shell simulation script

## Usage
Run `make sim` or `./sim/run_ghdl.sh` to compile and simulate with GHDL.
