# Project Overview: Counter ALU
- **Target**: Generic portable FPGA
- **Clock**: 100 MHz (10 ns period)
- **Reset**: Synchronous active-high
- **Scope**: 8-bit Arithmetic Logic Unit with registered outputs
- **Structure**: Top-level ALU, self-checking testbench, GHDL-ready scripts, constraints placeholder
- **Standards**: VHDL-2008, `ieee.std_logic_1164`, `ieee.numeric_std`
