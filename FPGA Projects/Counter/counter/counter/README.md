# Counter Project

## Description
This project implements an 8-bit ALU for FPGA synthesis and simulation.
It includes VHDL source files, a testbench, and GHDL simulation scripts.

## Structure
- `src/`: VHDL source files.
- `tb/`: Testbench files.
- `constraints/`: XDC constraints.
- `scripts/`: Simulation scripts.

## Usage
Run simulation: `make sim`
Clean: `make clean`
Help: `make help`
