# Mini CPU Core

A minimal single-cycle CPU core implemented in VHDL-2008 for educational purposes.

## Architecture

- **Data Width**: 8 bits
- **Registers**: 4 (r0-r3)
- **Program Memory**: 12-bit address space
- **Instruction Set**: NOP, LOAD, STORE, ADD, SUB, AND, OR, XOR, JMP, BEQ
- **Clock**: 100 MHz (configurable)
- **Reset**: Synchronous active-high
